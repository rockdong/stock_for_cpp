# 飞书消息回复优化 - 富文本消息设计

**日期**: 2026-03-14
**状态**: 已批准

## 目标

改善飞书股票机器人的消息交互体验，使用飞书富文本消息（卡片、按钮、菜单）替代纯文本展示。

## 消息类型映射

| 场景 | 当前形式 | 优化后形式 |
|------|----------|------------|
| 帮助信息 | 纯文本 | 富文本卡片 |
| 股票查询结果 | Markdown表格 | 富文本卡片（每行一个区块） |
| 图表生成确认 | 直接生成图片 | 交互卡片（带周期切换按钮） |
| 分析进度 | 纯文本+ASCII进度条 | 富文本卡片+按钮 |
| 分析结果 | Markdown表格 | 富文本卡片 |

## 详细设计

### 1. 富文本卡片格式

**帮助信息卡片**:
```
┌─────────────────────────────────┐
│  📖 股票分析机器人 - 帮助指南     │
├─────────────────────────────────┤
│                                 │
│  🔍 查询功能                     │
│  ─────────────────────────────  │
│  • 股票 <代码/名称> - 查询股票   │
│  • 股票列表 - 查看所有股票       │
│  • 行业 <行业名> - 按行业筛选    │
│                                 │
│  📊 分析功能                     │
│  ...                            │
└─────────────────────────────────┘
```

### 2. 交互卡片（带按钮）

**图表生成确认**:
- 显示股票信息和当前周期
- 提供日线/周线/月线切换按钮
- 提供刷新按钮

**分析进度**:
- 显示进度条、完成/失败计数
- 提供刷新、停止按钮

### 3. 按钮回调处理

用户点击按钮后，通过回调 message_id 识别操作，重新查询数据并更新消息。
